## 🔍 常见问题

### Q1: 服务无法启动

**可能原因**:
- 端口9091被占用
- 依赖包未安装

**解决方案**:
```bash
# 检查端口占用
netstat -ano | findstr "9091"

# 安装依赖
pip install fastapi uvicorn pydantic requests python-dotenv openai
```

---

### Q2: 测试连接失败

**错误信息**: `无法连接到服务`

**解决方案**:
1. 确认服务已启动（看到 `Uvicorn running on http://127.0.0.1:9091`）
2. 检查防火墙设置
3. 尝试访问 `http://127.0.0.1:9091/docs` 确认服务正常

---

### Q3: 时间冲突检测不准确

**可能原因**:
- 时间格式不正确
- 活动时间段设置不合理

**解决方案**:
- 使用标准时间格式：`HH:MM`（如 "09:00"）
- 或使用时间槽：`morning`, `afternoon`, `evening`
- 确保活动时间段之间有合理间隔（至少30分钟）

---

### Q4: 预算校验失败

**可能原因**:
- 费用项遗漏
- 旅行者数量未正确计算

**解决方案**:
- 检查 `structured_requirement` 中的 `traveler_count`
- 确认所有活动都有正确的价格字段
- 查看 `total_cost` 计算是否包含所有费用项

## 🛠️ API接口参考

### 1. 时间冲突检测

**接口**: `POST /api/v1/validation/time-conflict`

**请求体**:
```json
{
  "schedule": [...]
}
```

**响应**:
```json
{
  "code": 200,
  "msg": "时间冲突检测完成",
  "data": {
    "has_conflict": true/false,
    "conflicts": [...]
  }
}
```

---

### 2. 行程整合

**接口**: `POST /api/v1/integration/combine`

**请求体**:
```json
{
  "task_id": "xxx",
  "agent_results": {...},
  "structured_requirement": {...}
}
```

**响应**:
```json
{
  "code": 200,
  "msg": "行程整合成功",
  "data": {
    "task_id": "xxx",
    "day_plans": [...],
    "validation": {...},
    "total_cost": 1234.56
  }
}
```

---

### 3. 路线优化

**接口**: `POST /api/v1/integration/optimize-route`

**请求体**:
```json
{
  "attractions": [
    {"name": "景点A", "location": {"lat": 39.9, "lng": 116.4}},
    ...
  ]
}
```

**响应**:
```json
{
  "code": 200,
  "msg": "路线优化完成",
  "data": {
    "optimized_attractions": [...]
  }
}
```
