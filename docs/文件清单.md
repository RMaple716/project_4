## 📄 文件详细说明

### 🔧 核心脚本（3个）

#### 1. `scripts/sync_db.py` - 基础同步工具
**文件大小**: ~180 行  
**主要功能**:
- ✅ 自动检测并创建所有缺失的表
- ✅ 支持普通同步和强制重建两种模式
- ✅ 详细的表结构输出（列、索引、外键）
- ✅ 完整的执行日志和状态验证

**使用方法**:
```bash
python scripts/sync_db.py              # 普通同步
python scripts/sync_db.py --force      # 强制重建
```

**适用场景**:
- 首次初始化数据库
- 新增模型后同步
- 开发环境重置

---

#### 2. `scripts/db_migration.py` - 高级迁移工具
**文件大小**: ~250 行  
**主要功能**:
- ✅ 交互式菜单管理界面
- ✅ 查看所有表状态
- ✅ 检查数据库与代码定义的差异
- ✅ 单个表的创建/删除
- ✅ 列的添加/删除操作

**使用方法**:
```bash
python scripts/db_migration.py
# 然后选择相应选项进行操作
```

**菜单选项**:
```
1. 查看所有表状态
2. 检查表结构差异
3. 创建单个表
4. 删除单个表
5. 添加列
6. 删除列
7. 退出
```

**适用场景**:
- 精细化的表结构管理
- 调试和测试
- 学习数据库结构
