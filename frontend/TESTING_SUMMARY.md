## 🐛 常见问题速查

### Q1: npm install 很慢或失败？

**解决方法**：
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com
npm install

# 或清除缓存后重试
npm cache clean --force
npm install
```

---

### Q2: 端口被占用？

**错误信息**：
```
Port 3000 is in use
```

**解决方法**：
修改 `vite.config.ts`：
```typescript
server: {
  port: 3001,  // 改为其他端口
}
```

---

### Q3: 后端连接失败？

**检查清单**：
```bash
# 1. 检查后端是否运行
netstat -ano | findstr :9091

# 2. 如果没有输出，启动后端
python src/index.py

# 3. 检查防火墙设置
```

---

### Q4: TypeScript报错？

**错误信息**：
```
找不到模块"antd"或其相应的类型声明
```

**原因**：依赖未安装

**解决**：
```bash
npm install
```

安装完成后错误会自动消失。

---

### Q5: 提交后没有跳转？

**排查步骤**：
1. 检查Network标签的decompose请求
2. 查看响应状态码
3. 检查Console是否有JavaScript错误
4. 查看后端日志
